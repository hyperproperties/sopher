# Sopher
Hypercontracts in Go supporting HyperLTL contractual specifications for functions and HyperCTL* select constructs.

## Quickstart

```
go run ./cmd/main.go
# Or
make run
```
```
go build ./cmd/main.go
# Or
make build
```
```
go test ./...
# Or
make test
```
```
go fmt ./...
# Or
make fmt
```
